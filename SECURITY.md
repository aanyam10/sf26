# Security Policy

## Supported Versions

Only the latest `main` branch is supported for security updates.

## Reporting a Vulnerability

If you discover a security issue, do not open a public issue with exploit details.

Please report it privately to the repository owner/maintainer with:

- affected file(s)
- impact summary
- reproduction steps
- suggested mitigation (if available)

You should receive acknowledgment and next steps after review.
